# SalaryApp

A Java application that calculates employee salaries and bonuses.

## Features
- Calculate salaries based on hours worked
- Apply tax deductions automatically
- Generate monthly payroll report
- Supports multiple employees

## Technologies Used
- Java 17
- Maven (or Gradle)
- JUnit (for testing)

## How to Run
1. Clone the repository: `git clone https://github.com/CountToFiveXY/SalaryApp.git`
2. Open in your IDE (IntelliJ/Eclipse)
3. Run `Main.java`
4. (Optional) Run tests: `mvn test`
